## RawString

> swift5 이전버전에서 특수문자를 "\"를 사용해 나타내준 것과는 달리 swift5부터는 #와 "의 조합으로 특수문자 표현이 가능



10171번 - 고양이

``` swift
let test = #"""
\    /\
 )  ( ')
(  /  )
 \(__)|
"""#
     
print(test)
```



10172번 - 개

```swift
let test = #"""
|\_/|
|q p|   /}
( 0 )"""\
|"^"`    |
||_/=\\__|
"""#

print(test)swift
```



## Print

> ```swift
> print(_:separator:terminator:)
> ```



```swift
print("my", "name", "is", "zedd",separator : "...")
//my...name...is....zedd

for n in 1...5{
  print(n, terminator: " ")
}
//1 2 3 4 5
```
